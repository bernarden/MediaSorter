# Vima MediaSorter
A simple command-line tool that organizes media files (images and videos) into structured folders.

## Getting started

### Prerequisites
- .NET 10 SDK
- PowerShell (Required only for using the `Publish.ps1` script on Windows)

### Generate executable
- Windows: 
    - The self-contained executable can be generated by running the `\Scripts\Publish.ps1` script.
    - Alternativly run `dotnet publish Vima.MediaSorter -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true`

- Linux: 
    - The self-containedxecutable can be generated using `dotnet publish Vima.MediaSorter -c Release -r linux-x64 --self-contained true /p:PublishSingleFile=true`

### Run
The project can be run directly using `dotnet` or by running the genereted executable.

```bash
dotnet run --project Vima.MediaSorter -- -d "C:\path\to\media" -p IdentifyAndSortNewMedia
```

```bash
Vima.MediaSorter.exe -d "C:\path\to\media" -p IdentifyAndSortNewMedia
```

CLI options:
- `--directory`, `-d` - Directory to process. Defaults to the current working directory.
- `--processor`, `-p` - Select a processor to run directly (bypasses interactive menu). Valid values are defined by `ProcessorOption`.

## Disclaimer and Safety Warning
* **USE AT YOUR OWN RISK.** This software moves and modifies file paths. While designed to be safe, there is always a risk of data loss, corruption, or unexpected behavior.
* **Always Backup:** Before running the MediaSorter on any directory containing important or irreplaceable media, **ensure you have a complete and tested backup** of all files.
* **Test Small:** Strongly recommend running the software on a small, disposable test directory first to understand its behavior and verify its actions.
